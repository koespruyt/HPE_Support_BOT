# Contributing to HPE Support BOT

Thank you for your interest in contributing to the HPE Support BOT! We welcome contributions from the community and appreciate your efforts to enhance our project. Please follow these guidelines to ensure a smooth contribution process.

## How to Contribute

1. **Fork the Repository:**  
   Start by forking the repository to your own GitHub account. This will create a copy of the project that you can modify.

2. **Create a New Branch:**  
   Before making changes, create a new branch for your feature or bug fix. Use a descriptive name for your branch (e.g., `feature-add-new-endpoint` or `bugfix-fix-typo`).

   ```bash
   git checkout -b my-new-branch
   ```

3. **Make Your Changes:**  
   Work on the changes you want to implement. Be sure to follow any relevant coding standards and best practices.

4. **Commit Your Changes:**  
   Once you have made your changes, commit them with a clear and concise commit message that describes your changes.

   ```bash
   git commit -m "Description of changes"
   ```

5. **Push Changes to GitHub:**  
   Push your changes back to your forked repository on GitHub.

   ```bash
   git push origin my-new-branch
   ```

6. **Open a Pull Request:**  
   Navigate to the original repository and open a pull request (PR) from your new branch. Provide a detailed description of your changes and reference any related issues.

## Guidelines

- **Code Quality:** Ensure that your code is clean, well-structured, and follows the project's coding style.
- **Testing:** Include relevant tests for your changes. Run all existing tests to ensure nothing is broken.
- **Documentation:** Update any relevant documentation to reflect your changes.
- **Respect the Community:** Be respectful and supportive when interacting with other contributors. We strive for a positive and inclusive environment.

## Reporting Issues

If you encounter any issues or bugs, please report them by opening an issue in the repository, providing as much detail as possible.

Thank you for contributing to the HPE Support BOT!
